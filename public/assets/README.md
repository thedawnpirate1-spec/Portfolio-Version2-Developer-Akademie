# Assets

Statische Dateien, die direkt ausgeliefert werden. Im Code referenziert man sie
ab der Wurzel, z. B. `<img src="assets/images/profil.jpg">`.

- `images/` – Profilfoto, Projekt-Screenshots (Join, Pollo Loco, Simple CRM,
  Pokédex, Crypto Converter). **Hier dein eigenes Foto + deine eigenen
  Projektbilder ablegen** – das Template-Foto wird ersetzt.
- `icons/` – Skill-Icons (Angular, TypeScript, JavaScript, HTML, CSS, Git,
  Firebase, Rest API, Scrum, Material Design) sowie Social-Icons.
